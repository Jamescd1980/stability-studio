Copy this entire folder to your project location (e.g. Desktop/MyStoryboard).
Set stability-studio-mcp/config.yaml outputs.delivery to that folder path.

Subfolders:
  source/     - original reference uploads
  images/     - hero stills
  images/chain/ - I2V handoff frames
  assets/     - control maps (canny, lineart, openpose)
  clips/      - approved video clips
  audio/      - approved dialogue and SFX
  temp/       - raw Wan2GP/MOSS output pending approval
  logs/       - JSON run logs only
  final/      - assembled deliverables
  rejected/   - superseded attempts (optional)
